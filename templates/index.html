About Game Portal

Game Portal is a modern and interactive web-based gaming platform developed using Python Flask. It allows users to register, log in, play skill-based games, and track their performance in real time.

Game Portal is designed to improve logical thinking, memory power, and problem-solving skills through fun and engaging games. The platform is fully responsive and suitable for users of all age groups.
