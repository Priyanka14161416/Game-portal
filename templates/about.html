About Game Portal

Game Portal is a professional web-based gaming platform developed as using Python Flask. The platform provides multiple interactive games, real-time scoring, and user-friendly dashboards.

🎯 Project Objective

The main goal of this project is to design and develop a secure, scalable, and interactive gaming platform that demonstrates real-world web development skills using Python Flask.

🧠 Games Included

  • ✔ Tic Tac Toe (AI & Two Player)
  • ✔ Sudoku Game
  • ✔ Memory Match Game
  • ✔ Snake Game (Python)
  • ✔ Stack Game (Python)

⚙️ Technologies Used

  • Python Flask
  • HTML5, CSS3, JavaScript
  • SQLite Database
  • Flask-Login Authentication

🔐 Features

  • User Registration & Login
  • Admin & User Dashboard
  • Score Submission
  • Leaderboard System
  • Responsive UI

This project is developed for academic purposes and showcases practical implementation of full-stack web development concepts.
